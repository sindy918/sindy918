## Featured Projects

<details>
<summary><b>Anti-Cheat Monitoring & Activity Logging System</b></summary>

<br>

A Python-based activity monitoring system designed to track clipboard and keystroke behavior in real time, detect suspicious activity using rule-based logic, and generate timestamped alerts for review.

| Category | Details |
|---|---|
| Stack | Python, Logging, SMTP, Rule-Based Logic |
| Scale | 372-line Python tool with 15 functions and 1 class |
| Performance | Real-time clipboard and keystroke monitoring with configurable thresholds |
| Security | Alert handling, exception handling, input validation, activity logging |
| Impact | Improved monitoring reliability across multiple suspicious activity scenarios |
| Repository | [View Project](https://github.com/sindy918) |

The system includes a 500-character keystroke buffer, 30-character alert threshold, and 50% similarity threshold to detect suspicious behavior. SMTP email alerts and timestamped logs were integrated across startup, detection, file error, shutdown, and crash-handling scenarios.

</details>

<details>
<summary><b>Ethereum Fraud Detection System</b></summary>

<br>

A machine learning pipeline built to classify Ethereum transactions as normal or suspicious using data preprocessing, feature selection, class balancing, and supervised model evaluation.

| Category | Details |
|---|---|
| Stack | Python, Machine Learning, Data Preprocessing, Random Forest, XGBoost |
| Scale | 9,841 Ethereum transactions and 51-column transaction dataset |
| Performance | Achieved 99.15% accuracy and 99.95% ROC-AUC |
| Security | Fraud classification workflow for suspicious transaction detection |
| Impact | Improved identification of high-risk blockchain transaction patterns |
| Repository | [View Project](https://github.com/sindy918) |

The project involved cleaning 22,635 missing values, selecting 45 numeric features, applying SMOTE to balance the dataset, and evaluating Random Forest and XGBoost models on 3,065 test samples.

</details>
